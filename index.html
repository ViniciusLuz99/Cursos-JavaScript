Variáveis em JavaScript?

O que são variáveis em JavaScript?

Em JavaScript, as variáveis são usadas para armazenar valores. Você pode atribuir diferentes tipos de dados a uma variável, como números, strings, booleanos, objetos, arrays e muito mais.

Palavras-chave para declarar variáveis

Em JavaScript, existem três palavras-chave para declarar variáveis:

  • var: era a maneira antiga de declarar variáveis em JavaScript.
  • let: usado para declarar variáveis que podem ser alteradas posteriormente.
  • const: usado para declarar variáveis com valores constantes, que não podem ser alterados.

Exemplos de declaração de variáveis


let idade = 25;
let nome = "João";
const PI = 3.14;

Ao declarar uma variável, é uma boa prática atribuir um nome descritivo a ela. Os nomes de variáveis em JavaScript podem conter letras, números, sublinhados (_) e cifrões ($), mas devem começar com uma letra. Além disso, o JavaScript diferencia maiúsculas de minúsculas, portanto, idade e Idade são considerados nomes de variáveis diferentes.

Utilizando variáveis em expressões


let a = 10;
let b = 5;
let soma = a + b
let c = a > b;

Escopo das variáveis

As variáveis têm escopo, o que significa que elas podem ser acessadas apenas dentro do bloco em que foram declaradas.
Variáveis declaradas com var têm escopo de função, enquanto aquelas declaradas com let ou const têm escopo de bloco.

Operadores

Oque são operadores?

Operadores são símbolos especiais que realizam operações em valores. Eles podem ser aritméticos, de atribuição, de comparação ou lógicos.

Operadores Aritméticos

Os operadores aritméticos realizam operações matemáticas em valores numéricos. 

                  
let resultado = 10 + 5; // Soma: resultado é igual a 15
let resultado = 10 - 5; // Subtração: resultado é igual a 5
let resultado = 10 * 5; // Multiplicação: resultado é igual a 50
let resultado = 10 / 5; // Divisão: resultado é igual a 2
let resultado = 10 % 3; // Resto: resultado é igual a 1 (resto de 10 dividido por 3)

Operadores de Atribuição

Os operadores de atribuição são utilizados para atribuir valores a variáveis e realizar operações junto com a atribuição. 

                  
let numero = 5; // Atribuição simples
numero += 5; // Atribuição com operação: resultado será 10 (5 + 5)

Operadores de Comparação

Os operadores de comparação comparam valores e retornam um valor booleano (true ou false). 

                
console.log(5 == '5'); // Igual: true, pois ambos são iguais numericamente
console.log(5 != 10); // Não igual: true, pois 5 não é igual a 10
console.log(5 === '5'); // Estritamente igual: false, pois são de tipos diferentes (número e string)
console.log(5 !== '5'); // Estritamente não igual: true, pois são de tipos diferentes (número e string)
console.log(10 > 5); // Maior que: true, pois 10 é maior que 5

Operadores Lógicos

Os operadores lógicos são usados para combinar ou inverter valores booleanos. 

                
console.log(true && false); // E lógico: false, pois ambas as expressões não são verdadeiras
console.log(true || false); // Ou lógico: true, pois pelo menos uma das expressões é verdadeira
console.log(!true); // Não lógico: false, pois inverteu o valor true para false
Estruturas Condicionais

Oque são Estruturas condicionais?

As estruturas condicionais permitem ao programa tomar decisões com base em condições específicas.

Estrutura " if "

A estrutura "if"(O "if" pode ser traduzido como "se".)Ela é usada para verificar uma condição. Se a condição for verdadeira (ou seja, o resultado é verdadeiro ou "true"), o bloco de código dentro do "if" é executado. Caso contrário, o bloco de código é ignorado. 

                
// Exemplo com filme: Star Wars
let episodio = 4;

if (episodio === 4) {
 console.log("Que a força esteja com você!");
}

Estrutura "if-else"

A estrutura "if-else"(O "else" pode ser traduzido como "se não")Ele executa um bloco de código se a condição for verdadeira e outro bloco de código se a condição for falsa. 

                
        // Exemplo com jogo: Mario
let vidas = 2;

if (vidas > 0) {
  console.log("Você ainda está no jogo!");
} else {
  console.log("Game Over!");
}

Estrutura "else if"

A estrutura "else if"(O "else if" pode ser traduzido como "senão, se".) é usada quando há mais de uma condição a ser testada. 

                
let nota = 85;

if (nota >= 90) {
  console.log("Parabéns! Sua nota é A.");
} else if (nota >= 80) {
  console.log("Ótimo! Sua nota é B.");
} else if (nota >= 70) {
  console.log("Boa! Sua nota é C.");
} else {
  console.log("Estude mais. Sua nota é abaixo de C.");
}

Operador ternário

O operador ternário é uma forma abreviada de escrever uma estrutura condicional "if-else" em apenas uma linha. Ele é composto por três partes separadas por "?", e funciona da seguinte maneira:

condição ? valor_se_verdadeiro : valor_se_falso

Primeiro, é avaliada a condição. Se ela for verdadeira, o operador retorna o "valor_se_verdadeiro". Caso contrário, se a condição for falsa, o operador retorna o "valor_se_falso".

Vamos exemplificar com um caso relacionado à idade:

                
let idade = 20;
let mensagem = (idade >= 18) ? "Maior de idade" : "Menor de idade";

console.log(mensagem); // Resultado: "Maior de idade" (se idade for maior ou igual a 18) ou "Menor de idade" (caso contrário)
Funções

O que são funções?

As funções são blocos de código que agrupam tarefas específicas em um único lugar. Elas nos permitem reutilizar esse código várias vezes sem precisar repeti-lo.

Sintaxe básica de uma função:

                
function nomeDaFuncao() {
// Bloco de código da função
// Aqui você escreve as ações que a função irá executar
}

Exemplo de função que exibe uma mensagem:

                
function saudacao() {
  console.log("Olá!");
}

// Chamando a função para exibir a mensagem
saudacao(); // Saída: Olá!

Funções com parâmetros:

As funções podem receber valores, chamados de parâmetros, que serão usados dentro do bloco de código. 

                
function saudacao(nome) {
  console.log(`Olá, ${nome}!`);
}

// Chamando a função com um valor específico para o parâmetro
saudacao("Link"); // Saída: Olá, Link!

Funções com retorno:

As funções também podem retornar valores para serem utilizados fora da função. 

                
function personagemPrincipal(jogo) {
  if (jogo === "Super Mario") {
    return "Mario";
  } else if (jogo === "The Legend of Zelda") {
    return "Link";
  } else {
    return "Personagem não identificado";
  }
}
// Chamando a função e atribuindo o resultado a uma variável
let personagem = personagemPrincipal("The Legend of Zelda");
console.log(personagem); // Saída: Link

Arrow Functions (Funções de seta)

As Arrow Functions são uma forma mais concisa de escrever funções em JavaScript. Elas são especialmente úteis quando precisamos de funções curtas e diretas.

Vantagens das Arrow Functions:

  • Sintaxe mais curta e simplificada;
  • O valor de "this" é herdado do contexto em que a Arrow Function foi definida;
  • Torna o código mais enxuto e legível.
    
// Função tradicional para verificar a quantidade de moedas do Mario
function verificarMoedas(moedas) {
  if (moedas >= 100) {
    console.log("Você coletou muitas moedas! Parabéns!");
  } else {
    console.log("Continue coletando moedas para aumentar sua pontuação!");
  }
}

// Com Arrow Function, podemos simplificar o código
let verificarMoedas = (moedas) => {
  if (moedas >= 100) {
    console.log("Você coletou muitas moedas! Parabéns!");
  } else {
    console.log("Continue coletando moedas para aumentar sua pontuação!");
  }
};

// Chamando a função com um valor de moedas para o Mario
verificarMoedas(120);

As Arrow Functions são uma ferramenta poderosa que nos permite escrever código mais enxuto e legível. Elas são amplamente utilizadas em programação moderna e podem trazer mais eficiência e clareza ao nosso código.

Funções anônimas (Anonymous Functions)

As funções anônimas, também conhecidas como "Anonymous Functions" em inglês, são funções que não possuem um nome definido. Elas são usadas quando não precisamos reutilizar a função em outras partes do código, ou quando queremos escrever uma função de forma mais concisa.

Existem duas formas de criar funções anônimas:

1. Expressão de função anônima

                
// Criando uma função anônima e atribuindo-a a uma variável
let saudacao = function(nome) {
  return `Olá, ${nome}!`;
};

// Chamando a função
console.log(saudacao("Samus")); // Saída: Olá, Samus!

2. Arrow Function (Função de seta)

                
// Arrow function com um único parâmetro
let saudacao = (nome) => {
  return `Olá, ${nome}!`;
};

// Quando a função possui apenas uma linha de código, podemos omitir as chaves e o "return"
// O valor retornado é implícito
let saudacao = (nome) => `Olá, ${nome}!`;

// Chamando a função
console.log(saudacao("Samus")); // Saída: Olá, Samus!

Estruturas de repetição:

As estruturas de repetição permitem que você execute um bloco de código várias vezes. A declaração for é usada quando você sabe exatamente quantas vezes deseja repetir o código. A declaração while é usada quando você deseja repetir o código enquanto uma condição for verdadeira. A declaração do...while é semelhante ao while, mas garante que o código seja executado pelo menos uma vez.

Exemplo:

    
for (let i = 0; i < 5; i++) {
  console.log(i);
}

let contador = 0;
while (contador < 5) {
  console.log(contador);
  contador++;
}

let valor = 5;
do {
  console.log(valor);
  valor--;
} while (valor > 0);

Ensinamento: As estruturas de repetição são úteis quando você precisa executar um bloco de código várias vezes. Certifique-se de definir as condições corretas para evitar loops infinitos e garantir que o código seja executado o número correto de vezes.

Arrays:

Arrays são estruturas de dados que permitem armazenar e acessar vários valores em uma única variável. Os elementos de um array são indexados numericamente, começando em 0.

Exemplo:

    
let numeros = [1, 2, 3, 4, 5];
console.log(numeros[2]); // Output: 3

numeros.push(6);
console.log(numeros); // Output: [1, 2, 3, 4, 5, 6]

Ensinamento: Arrays são úteis para armazenar coleções de valores relacionados. Aprenda a criar, acessar e modificar elementos em um array, além de usar métodos embutidos para manipulá-los.

Objetos:

Objetos são estruturas de dados que permitem armazenar propriedades e métodos relacionados em uma única variável. Cada propriedade é composta por uma chave e um valor associado.

Exemplo:

    
let pessoa = {
  nome: "João",
  idade: 30,
  profissao: "Desenvolvedor"
};

console.log(pessoa.nome); // Output: João

Ensinamento: Objetos são úteis para representar entidades complexas com propriedades e comportamentos. Aprenda a criar objetos, acessar e modificar propriedades, e entender o conceito de referência em objetos.
