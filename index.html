Variáveis

O que são variáveis em JavaScript?

Em JavaScript, as variáveis são usadas para armazenar valores. Você pode atribuir diferentes tipos de dados a uma variável, como números, strings, booleanos, objetos, arrays e muito mais.

Palavras-chave para declarar variáveis

Em JavaScript, existem três palavras-chave para declarar variáveis:

  • var: era a maneira antiga de declarar variáveis em JavaScript.
  • let: usado para declarar variáveis que podem ser alteradas posteriormente.
  • const: usado para declarar variáveis com valores constantes, que não podem ser alterados.

Exemplos de declaração de variáveis


let idade = 25;
let nome = "João";
const PI = 3.14;

Ao declarar uma variável, é uma boa prática atribuir um nome descritivo a ela. Os nomes de variáveis em JavaScript podem conter letras, números, sublinhados (_) e cifrões ($), mas devem começar com uma letra. Além disso, o JavaScript diferencia maiúsculas de minúsculas, portanto, idade e Idade são considerados nomes de variáveis diferentes.

Utilizando variáveis em expressões


let a = 10;
let b = 5;
let soma = a + b
let c = a > b;

Escopo das variáveis

As variáveis têm escopo, o que significa que elas podem ser acessadas apenas dentro do bloco em que foram declaradas.
Variáveis declaradas com var têm escopo de função, enquanto aquelas declaradas com let ou const têm escopo de bloco.

Operadores

Oque são operadores?

Operadores são símbolos especiais que realizam operações em valores. Eles podem ser aritméticos, de atribuição, de comparação ou lógicos.

Operadores Aritméticos

Os operadores aritméticos realizam operações matemáticas em valores numéricos. 

                  
let resultado = 10 + 5; // Soma: resultado é igual a 15
let resultado = 10 - 5; // Subtração: resultado é igual a 5
let resultado = 10 * 5; // Multiplicação: resultado é igual a 50
let resultado = 10 / 5; // Divisão: resultado é igual a 2
let resultado = 10 % 3; // Resto: resultado é igual a 1 (resto de 10 dividido por 3)

Operadores de Atribuição

Os operadores de atribuição são utilizados para atribuir valores a variáveis e realizar operações junto com a atribuição. 

                  
let numero = 5; // Atribuição simples
numero += 5; // Atribuição com operação: resultado será 10 (5 + 5)

Operadores de Comparação

Os operadores de comparação comparam valores e retornam um valor booleano (true ou false). 

                
console.log(5 == '5'); // Igual: true, pois ambos são iguais numericamente
console.log(5 != 10); // Não igual: true, pois 5 não é igual a 10
console.log(5 === '5'); // Estritamente igual: false, pois são de tipos diferentes (número e string)
console.log(5 !== '5'); // Estritamente não igual: true, pois são de tipos diferentes (número e string)
console.log(10 > 5); // Maior que: true, pois 10 é maior que 5

Operadores Lógicos

Os operadores lógicos são usados para combinar ou inverter valores booleanos. 

                
console.log(true && false); // E lógico: false, pois ambas as expressões não são verdadeiras
console.log(true || false); // Ou lógico: true, pois pelo menos uma das expressões é verdadeira
console.log(!true); // Não lógico: false, pois inverteu o valor true para false
Estruturas Condicionais

Oque são Estruturas condicionais?

As estruturas condicionais permitem ao programa tomar decisões com base em condições específicas.

Estrutura " if "

A estrutura "if"(O "if" pode ser traduzido como "se".)Ela é usada para verificar uma condição. Se a condição for verdadeira (ou seja, o resultado é verdadeiro ou "true"), o bloco de código dentro do "if" é executado. Caso contrário, o bloco de código é ignorado. 

                
// Exemplo com filme: Star Wars
let episodio = 4;

if (episodio === 4) {
 console.log("Que a força esteja com você!");
}

Estrutura "if-else"

A estrutura "if-else"(O "else" pode ser traduzido como "se não")Ele executa um bloco de código se a condição for verdadeira e outro bloco de código se a condição for falsa. 

                
// Exemplo com jogo: Mario
let vidas = 2;

if (vidas > 0) {
  console.log("Você ainda está no jogo!");
} else {
  console.log("Game Over!");
}

Estrutura "else if"

A estrutura "else if"(O "else if" pode ser traduzido como "senão, se".) é usada quando há mais de uma condição a ser testada. 

                
let nota = 85;

if (nota >= 90) {
  console.log("Parabéns! Sua nota é A.");
} else if (nota >= 80) {
  console.log("Ótimo! Sua nota é B.");
} else if (nota >= 70) {
  console.log("Boa! Sua nota é C.");
} else {
  console.log("Estude mais. Sua nota é abaixo de C.");
}

Operador ternário

O operador ternário é uma forma abreviada de escrever uma estrutura condicional "if-else" em apenas uma linha. Ele é composto por três partes separadas por "?", e funciona da seguinte maneira:

condição ? valor_se_verdadeiro : valor_se_falso

Primeiro, é avaliada a condição. Se ela for verdadeira, o operador retorna o "valor_se_verdadeiro". Caso contrário, se a condição for falsa, o operador retorna o "valor_se_falso".

Vamos exemplificar com um caso relacionado à idade:

                
let idade = 20;
let mensagem = (idade >= 18) ? "Maior de idade" : "Menor de idade";

console.log(mensagem);
// Resultado: "Maior de idade" (se idade for maior ou igual a 18) ou "Menor de idade" (caso contrário)
Funções

O que são funções?

As funções são blocos de código que agrupam tarefas específicas em um único lugar. Elas nos permitem reutilizar esse código várias vezes sem precisar repeti-lo.

Sintaxe básica de uma função:

                
function nomeDaFuncao() {
// Bloco de código da função
// Aqui você escreve as ações que a função irá executar
}

Exemplo de função que exibe uma mensagem:

                
function saudacao() {
  console.log("Olá!");
}

// Chamando a função para exibir a mensagem
saudacao(); // Saída: Olá!

Funções com parâmetros:

As funções podem receber valores, chamados de parâmetros, que serão usados dentro do bloco de código. 

                
function saudacao(nome) {
  console.log(`Olá, ${nome}!`);
}

// Chamando a função com um valor específico para o parâmetro
saudacao("Link"); // Saída: Olá, Link!

Funções com retorno:

As funções também podem retornar valores para serem utilizados fora da função. 

                
function personagemPrincipal(jogo) {
  if (jogo === "Super Mario") {
    return "Mario";
  } else if (jogo === "The Legend of Zelda") {
    return "Link";
  } else {
    return "Personagem não identificado";
  }
}
// Chamando a função e atribuindo o resultado a uma variável
let personagem = personagemPrincipal("The Legend of Zelda");
console.log(personagem); // Saída: Link

Arrow Functions (Funções de seta)

As Arrow Functions são uma forma mais concisa de escrever funções em JavaScript. Elas são especialmente úteis quando precisamos de funções curtas e diretas.

Vantagens das Arrow Functions:

  • Sintaxe mais curta e simplificada;
  • O valor de "this" é herdado do contexto em que a Arrow Function foi definida;
  • Torna o código mais enxuto e legível.
    
// Função tradicional para verificar a quantidade de moedas do Mario
function verificarMoedas(moedas) {
  if (moedas >= 100) {
    console.log("Você coletou muitas moedas! Parabéns!");
  } else {
    console.log("Continue coletando moedas para aumentar sua pontuação!");
  }
}

// Com Arrow Function, podemos simplificar o código
let verificarMoedas = (moedas) => {
  if (moedas >= 100) {
    console.log("Você coletou muitas moedas! Parabéns!");
  } else {
    console.log("Continue coletando moedas para aumentar sua pontuação!");
  }
};

// Chamando a função com um valor de moedas para o Mario
verificarMoedas(120);

As Arrow Functions são uma ferramenta poderosa que nos permite escrever código mais enxuto e legível. Elas são amplamente utilizadas em programação moderna e podem trazer mais eficiência e clareza ao nosso código.

Funções anônimas (Anonymous Functions)

As funções anônimas, também conhecidas como "Anonymous Functions" em inglês, são funções que não possuem um nome definido. Elas são usadas quando não precisamos reutilizar a função em outras partes do código, ou quando queremos escrever uma função de forma mais concisa.

Existem duas formas de criar funções anônimas:

1. Expressão de função anônima

                
// Criando uma função anônima e atribuindo-a a uma variável
let saudacao = function(nome) {
  return `Olá, ${nome}!`;
};

// Chamando a função
console.log(saudacao("Samus")); // Saída: Olá, Samus!

2. Arrow Function (Função de seta)

                
// Arrow function com um único parâmetro
let saudacao = (nome) => {
  return `Olá, ${nome}!`;
};

// Quando a função possui apenas uma linha de código, podemos omitir as chaves e o "return"
// O valor retornado é implícito
let saudacao = (nome) => `Olá, ${nome}!`;

// Chamando a função
console.log(saudacao("Samus")); // Saída: Olá, Samus!
Arrays

O que é um array?

Um array é uma lista ordenada de elementos em JavaScript. Cada elemento em um array tem uma posição chamada índice, que começa em 0. Os arrays podem armazenar diferentes tipos de dados, como números, strings e objetos.

Criando um array

Para criar um array, utilizamos colchetes [] e separamos os elementos por vírgula. 

    
const frutas = ['maçã', 'banana', 'laranja'];

Acessando elementos

Podemos acessar elementos de um array utilizando seu índice entre colchetes. 

    
const frutas = ['maçã', 'banana', 'laranja'];
console.log(frutas[0]); // Saída: 'maçã'
console.log(frutas[1]); // Saída: 'banana'
console.log(frutas[2]); // Saída: 'laranja'

Percorrendo um array

Podemos percorrer os elementos de um array usando um loop, como o for ou o forEach(). 

    
const frutas = ['maçã', 'banana', 'laranja'];
for (let i = 0; i < frutas.length; i++) {
  console.log(frutas[i]);
}

Exemplo com forEach()

Outra forma de percorrer um array é usando o método forEach(). 

    
const frutas = ['maçã', 'banana', 'laranja'];
frutas.forEach(function(fruta) {
  console.log(fruta);
});
Manipulação de Arrays

Manipulação Básica de Arrays:

push(element): Adiciona um elemento ao final do array.
pop(): Remove e retorna o último elemento do array.
unshift(element): Adiciona um elemento no início do array.
shift(): Remove e retorna o primeiro elemento do array. 

                  
const arrayPush = [1, 2, 3];
arrayPush.push(4);
console.log(arrayPush); // Saída: [1, 2, 3, 4]

const arrayPop = [1, 2, 3];
const poppedElement = arrayPop.pop();
console.log(poppedElement); // Saída: 3

const arrayUnshift = [2, 3, 4];
arrayUnshift.unshift(1);
console.log(arrayUnshift); // Saída: [1, 2, 3, 4]

const arrayShift = [1, 2, 3];
const shiftedElement = arrayShift.shift();
console.log(shiftedElement); // Saída: 1

Acesso e Iteração:

length: Retorna o número de elementos no array.
forEach(callback): Executa uma função em cada elemento do array.
map(callback): Cria um novo array com resultados de uma função aplicada a cada elemento.
filter(callback): Cria um novo array com elementos que passam em um teste. 

                  
const arrayLength = [1, 2, 3, 4, 5];
console.log(arrayLength); // Saída: 5

const arrayForEach = [1, 2, 3];
arrayForEach.forEach(item => {
    console.log(item * 2);
});

const arrayMap = [1, 2, 3];
const doubledArray = arrayMap.map(item => item * 2);
console.log(doubledArray);

const arrayFilter = [1, 2, 3, 4, 5, 6];
const evenNumbers = arrayFilter.filter(item => item % 2 === 0);
console.log(evenNumbers);

Ordenação e Busca:

sort(): Ordena os elementos do array (pode exigir uma função de comparação).
reverse(): Inverte a ordem dos elementos no array.
indexOf(element): Retorna o índice da primeira ocorrência do elemento.
find(callback): Retorna o primeiro elemento que atende a um critério.
findIndex(callback): Retorna o índice do primeiro elemento que atende a um critério. 

                  
const arraySort = [3, 1, 4, 2];
arraySort.sort();
console.log(arraySort); // Saída: [1, 2, 3, 4]

const arrayReverse = [1, 2, 3, 4];
arrayReverse.reverse();
console.log(arrayReverse); // Saída: [4, 3, 2, 1]

const arrayIndexOf = [10, 20, 30, 40];
const index = arrayIndexOf.indexOf(30);
console.log(index); // Saída: 2

const arrayFind = [10, 20, 30, 40];
const foundElement = arrayFind.find(item => item > 20);
console.log(foundElement); // Saída: 30

const arrayFindIndex = [10, 20, 30, 40];
const foundIndex = arrayFindIndex.findIndex(item => item > 20);
console.log(foundIndex); // Saída: 2

Transformação e Redução:

reduce(callback, initialValue): Aplica uma função a um acumulador e a cada elemento, reduzindo o array a um único valor.
reduceRight(callback, initialValue): Semelhante ao reduce, mas começa pelo último elemento.
slice(start, end): Retorna uma parte do array, definida por índices de início e fim. 

                  
const arrayReduce = [1, 2, 3, 4, 5];
const sum = arrayReduce.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
console.log(sum); // Saída: 15

const arrayReduceRight = [1, 2, 3, 4, 5];
const reversedConcatenation = arrayReduceRight.reduceRight((accumulator, currentValue)=> accumulator + currentValue, '');
console.log(reversedConcatenation); // Saída: "54321"

const arraySlice = [1, 2, 3, 4, 5];
cnst slicedArray = arraySlice.slice(1, 4);
console.log(slicedArray); // Saída: [2, 3, 4]

Verificação de Elementos:

every(callback): Verifica se todos os elementos atendem a um critério.
some(callback): Verifica se pelo menos um elemento atende a um critério.
includes(element): Verifica se o array contém um determinado elemento. 

                  
const arrayEvery = [2, 4, 6, 8];
const allEven = arrayEvery.every(item => item % 2 === 0);
console.log(allEven); // Saída: true

const arraySome = [1, 3, 5, 6];
const hasEven = arraySome.some(item => item % 2 === 0);
console.log(hasEven); // Saída: true

const arrayIncludes = [10, 20, 30, 40];
const hasValue = arrayIncludes.includes(30);
console.log(hasValue); // Saída: true

Transformação Avançada:

flatMap(callback): Aplica uma função a cada elemento e, em seguida, aplana os resultados em um único array.
from(arrayLike, callback): Converte array-like objects (como NodeList) em arrays. 

                  
const arrayFlatMap = [1, 2, 3];
const mappedAndFlattened = arrayFlatMap.flatMap(item => [item * 2, item * 3]);
console.log(mappedAndFlattened); // Saída: [2, 3, 4, 6, 6, 9]

const nodeList = document.querySelectorAll('p'); // Exemplo de NodeList
const arrayFromNodeList = Array.from(nodeList);
console.log(arrayFromNodeList); // Saída: Array contendo elementos do NodeList
Objetos

Introdução aos Objetos:

Em JavaScript, os objetos são estruturas que nos permitem armazenar e organizar dados relacionados. Eles são como caixas que podem conter várias informações juntas. Vamos imaginar um objeto como um baú mágico que guarda diferentes tesouros.

Exemplo do Objeto "Jogador":

Imagine que estamos criando um jogo de aventura com um jogador. Vamos definir um objeto chamado "jogador" que terá algumas propriedades, como nome, nível e pontos. 


const jogador = {
  nome: "Aventureiro",
  nivel: 5,
  pontos: 200
};

Podemos acessar as propriedades do objeto usando a notação de ponto. É como abrir o baú e ver o que está dentro. 


console.log(jogador.nome); // Saída: "Aventureiro"
console.log(jogador.nivel); // Saída: 5
console.log(jogador.pontos); // Saída: 200

Arrays dentro de Objetos:

Arrays são listas de itens em JavaScript. Podemos colocar um array dentro de um objeto para representar itens que o jogador possui. 


const jogador = {
  nome: "Aventureiro",
  nivel: 5,
  pontos: 200,
  itens: ["espada", "escudo", "poção de cura"]
};

Vamos abrir o baú "itens" e ver quais equipamentos o jogador possui. 


console.log(jogador.itens[0]); // Saída: "espada"
console.log(jogador.itens[1]); // Saída: "escudo"
console.log(jogador.itens[2]); // Saída: "poção de cura"

Funções dentro de Objetos:

Além das propriedades, podemos adicionar funções ao objeto. Essas funções são chamadas de métodos do objeto.

No entanto, é importante evitar o uso de Arrow Functions para definir métodos, pois o "this" dentro de uma Arrow Function não se refere ao próprio objeto em que ela está sendo utilizada."

"O "this" é uma referência especial em JavaScript que se refere ao objeto em que a função está sendo executada. No caso dos métodos de objetos, o "this" aponta para o próprio objeto que contém o método 


const jogador = {
  nome: "Aventureiro",
  nivel: 5,
  pontos: 200,
  itens: ["espada", "escudo", "poção de cura"],
  atacarMonstro: function(monstro) {
    console.log(this.nome + " atacou o monstro " + monstro + "!");
  }
};

Agora, vamos chamar o método "atacarMonstro" e ver o jogador em ação! 


jogador.atacarMonstro("Dragão"); 
// Saída: "Aventureiro atacou o monstro Dragão!"
Loops

For Loop:

O loop for é usado quando sabemos exatamente quantas vezes queremos repetir um bloco de código. 

      
for (let i = 0; i < 5; i++) {
  console.log(i); // Output: 0, 1, 2, 3, 4
}

While Loop:

O loop while é usado quando queremos repetir um bloco de código enquanto uma condição for verdadeira. 

      
let count = 0;
while (count < 5) {
  console.log(count); // Output: 0, 1, 2, 3, 4
  count++;
}

Do-While Loop:

O loop do-while é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez, mesmo se a condição inicial for falsa. 

      
let num = 5;
do {
  console.log(num); // Output: 5 (executa pelo menos uma vez)
  num++;
} while (num < 5);

For...of Loop:

O loop for...of é usado para iterar sobre elementos de um objeto iterável (como arrays e strings). 

      
const colors = ['red', 'green', 'blue'];
for (const color of colors) {
  console.log(color); // Output: 'red', 'green', 'blue'
}

For...in Loop:

O loop for...in é usado para iterar sobre propriedades de um objeto. 

      
const person = { name: 'Alice', age: 30, job: 'engineer' };
for (const key in person) {
  console.log(`${key}: ${person[key]}`);
  // Output: 'name: Alice', 'age: 30', 'job: engineer'
}
