DOM(Document Object Model)

O que é o DOM:

Modelo de representação hierárquica de documentos HTML/XML. Permite manipular a estrutura e conteúdo de páginas web.

Árvore DOM:

Organiza elementos da página em uma árvore. Cada elemento é um "nó" no DOM.

Acesso a Elementos:

  • document.querySelector para selecionar elementos por seletor CSS.

  • document.getElementById para selecionar por ID.

  • document.getElementsByClassName para selecionar por classe. 

                        
const elemento = document.querySelector("#meuElemento");
elemento.textContent = "Novo conteúdo";

Manipulação de Conteúdo:

Mudar texto: elemento.textContent = "Novo texto".

Mudar HTML interno: elemento.innerHTML = "<p>Novo parágrafo</p>". 

                        
const elemento = document.querySelector("#meuElemento");
elemento.innerHTML = "<p>Novo parágrafo</p>";

Manipulação de Atributos:

Mudar atributos: elemento.setAttribute("atributo", "valor")

Obter valor de atributo: elemento.getAttribute("atributo"). 

                        
const elemento = document.querySelector("#meuElemento");
elemento.setAttribute("data-id", "123");
const valor = elemento.getAttribute("data-id");
console.log(valor); // Saída: "123"

Eventos:

Adicionar eventos: elemento.addEventListener("evento", função). 

                        
const botao = document.querySelector("#meuBotao");
botao.addEventListener("click", () => {
    console.log("Botão clicado!");
});

Criação de Elementos:

Criar elemento: document.createElement("tag").

Adicionar ao DOM: elementoPai.appendChild(novoElemento). 

                        
const novoParagrafo = document.createElement("p");
novoParagrafo.textContent = "Novo parágrafo criado!";
const elementoPai = document.querySelector("#meuElementoPai");
elementoPai.appendChild(novoParagrafo);

Remoção de Elementos:

Remover elemento: elementoPai.removeChild(elementoFilho). 

                        
const elementoPai = document.querySelector("#meuElementoPai");
const elementoFilho = document.querySelector("#meuElementoFilho");
elementoPai.removeChild(elementoFilho);

Navegação no DOM:

Acessar elementos pai: elemento.parentNode. Acessar elementos filhos: elemento.childNodes. Acessar próximo elemento irmão: elemento.nextSibling. Acessar elemento irmão anterior: elemento.previousSibling. 

                        
const filho = document.querySelector("#filho");
const pai = filho.parentNode;
const proximoIrmão = filho.nextSibling;
const irmãoAnterior = filho.previousSibling;

Modificação de Estilos:

Mudar estilo: elemento.style.propriedade = "valor". 

                        
const elemento = document.querySelector("#meuElemento");
elemento.style.color = "blue";
Selecionar Elementos no DOM

Seleção com querySelector

document.querySelector Permite selecionar um elemento usando um seletor CSS. Retorna o primeiro elemento encontrado que corresponde ao seletor. Útil para seleções específicas e únicas. 

            
// Exemplo:
const elemento = document.querySelector("#idDoElemento");

Seleção querySelectorALL:

document.querySelectorAll Seleciona todos os elementos que correspondem ao seletor CSS. Retorna uma lista de elementos (NodeList). Permite iterar sobre os elementos selecionados. Útil quando há vários elementos correspondentes. 

            
// Exemplo:
const elementos = document.querySelectorAll(".classeDoElemento");
elementos.forEach(elemento => {
    // Faça algo com cada elemento
});

Seleção com document.getElementById e document.getElementsByClassName:

document.getElementById seleciona um elemento por ID único.

document.getElementsByClassName seleciona elementos por classe.

Ambos retornam o próprio elemento ou uma coleção de elementos (HTMLCollection). Menos flexíveis, mas ainda úteis em certos casos. 

            
// Exemplos:
const elementoPorId = document.getElementById("idDoElemento");
const elementosPorClasse = document.getElementsByClassName("classeDoElemento");

Dicas Adicionais:

  • Use IDs para seleção rápida de elementos únicos.
  • Use classes para selecionar grupos de elementos com o mesmo estilo/comportamento.
  • Use querySelector e querySelectorAll para seleções flexíveis e poderosas.

Exercício Prático:

Vamos supor que você deseja alterar o texto de um parágrafo com o ID "meuParagrafo" para "Olá, Mundo!". Use document.querySelector para realizar essa tarefa. 

            
// Seleciona o elemento por ID e muda o texto
const paragrafo = document.querySelector("#meuParagrafo");
paragrafo.textContent = "Olá, Mundo!";
