DOM(Document Object Model)

O que é o DOM:

Modelo de representação hierárquica de documentos HTML/XML. Permite manipular a estrutura e conteúdo de páginas web.

Árvore DOM:

Organiza elementos da página em uma árvore. Cada elemento é um "nó" no DOM.

Acesso a Elementos:

  • document.querySelector para selecionar elementos por seletor CSS.

  • document.getElementById para selecionar por ID.

  • document.getElementsByClassName para selecionar por classe. 

                        
const elemento = document.querySelector("#meuElemento");
elemento.textContent = "Novo conteúdo";

Manipulação de Conteúdo:

Mudar texto: elemento.textContent = "Novo texto".

Mudar HTML interno: elemento.innerHTML = "<p>Novo parágrafo</p>". 

                        
const elemento = document.querySelector("#meuElemento");
elemento.innerHTML = "<p>Novo parágrafo</p>";

Manipulação de Atributos:

Mudar atributos: elemento.setAttribute("atributo", "valor")

Obter valor de atributo: elemento.getAttribute("atributo"). 

                        
const elemento = document.querySelector("#meuElemento");
elemento.setAttribute("data-id", "123");
const valor = elemento.getAttribute("data-id");
console.log(valor); // Saída: "123"

Eventos:

Adicionar eventos: elemento.addEventListener("evento", função). 

                        
const botao = document.querySelector("#meuBotao");
botao.addEventListener("click", () => {
    console.log("Botão clicado!");
});

Criação de Elementos:

Criar elemento: document.createElement("tag").

Adicionar ao DOM: elementoPai.appendChild(novoElemento). 

                        
const novoParagrafo = document.createElement("p");
novoParagrafo.textContent = "Novo parágrafo criado!";
const elementoPai = document.querySelector("#meuElementoPai");
elementoPai.appendChild(novoParagrafo);

Remoção de Elementos:

Remover elemento: elementoPai.removeChild(elementoFilho). 

                        
const elementoPai = document.querySelector("#meuElementoPai");
const elementoFilho = document.querySelector("#meuElementoFilho");
elementoPai.removeChild(elementoFilho);

Navegação no DOM:

Acessar elementos pai: elemento.parentNode. Acessar elementos filhos: elemento.childNodes. Acessar próximo elemento irmão: elemento.nextSibling. Acessar elemento irmão anterior: elemento.previousSibling. 

                        
const filho = document.querySelector("#filho");
const pai = filho.parentNode;
const proximoIrmão = filho.nextSibling;
const irmãoAnterior = filho.previousSibling;

Modificação de Estilos:

Mudar estilo: elemento.style.propriedade = "valor". 

                        
const elemento = document.querySelector("#meuElemento");
elemento.style.color = "blue";
Selecionar Elementos no DOM

Seleção com querySelector

document.querySelector Permite selecionar um elemento usando um seletor CSS. Retorna o primeiro elemento encontrado que corresponde ao seletor. Útil para seleções específicas e únicas. 

            
const elemento = document.querySelector("#idDoElemento");

Seleção querySelectorALL:

document.querySelectorAll Seleciona todos os elementos que correspondem ao seletor CSS. Retorna uma lista de elementos (NodeList). Permite iterar sobre os elementos selecionados. Útil quando há vários elementos correspondentes. 

            
const elementos = document.querySelectorAll(".classeDoElemento");
elementos.forEach(elemento => {
    // Faça algo com cada elemento
});

Seleção com getElementById e getElementsByClassName:

document.getElementById seleciona um elemento por ID único.

document.getElementsByClassName seleciona elementos por classe.

Ambos retornam o próprio elemento ou uma coleção de elementos (HTMLCollection). Menos flexíveis, mas ainda úteis em certos casos. 

            
const elementoPorId = document.getElementById("idDoElemento");
const elementosPorClasse = document.getElementsByClassName("classeDoElemento");

Dicas Adicionais:

  • Use IDs para seleção rápida de elementos únicos.
  • Use classes para selecionar grupos de elementos com o mesmo estilo/comportamento.
  • querySelector e querySelectorAll para seleções flexíveis e poderosas.

Exercício Prático:

Vamos supor que você deseja alterar o texto de um parágrafo com o ID "meuParagrafo" para "Olá, Mundo!". Use document.querySelector para realizar essa tarefa. 

            
// Seleciona o elemento por ID e muda o texto
const paragrafo = document.querySelector("#meuParagrafo");
paragrafo.textContent = "Olá, Mundo!";
Eventos de Clique DOM

Eventos no DOM

Eventos no Document Object Model (DOM) permitem que você crie interatividade em páginas web, respondendo a ações do usuário, como cliques, pressionamentos de teclas, passagem do mouse, entre outros. Vamos explorar os conceitos-chave:

  • O que são eventos: Eventos são ações que ocorrem no navegador enquanto o usuário interage com uma página. Eles podem ser acionados por ações do usuário ou pelo próprio navegador (ex: carregamento da página).

  • Selecionando elementos: Para trabalhar com eventos, primeiro, selecione o elemento HTML ao qual deseja adicionar um evento. Você pode usar métodos como getElementById, querySelector, ou selecionar diretamente por nome de tag, classe ou outros atributos.

  • Adicionando um Event Listener: Use o método addEventListener para registrar um "ouvinte de evento" no elemento selecionado. Esse ouvinte aguardará o evento especificado e executará uma função quando o evento ocorrer.

  • Função do Evento: A função do evento é o código que será executado quando o evento acontecer. Essa função pode realizar ações específicas, alterar o conteúdo da página, modificar estilos, interagir com APIs, etc..

Eventos de Clique no DOM

Os eventos de clique são uma parte fundamental da interatividade em páginas da web. Eles permitem que você execute ações quando um elemento HTML é clicado pelo usuário. 

                
// Exemplo de evento de clique
const botao = document.getElementById("meuBotao");
botao.addEventListener("click", () => {
  alert("Você clicou no botão!");
  botao.innerText = "Clicado";
  botao.style.backgroundColor = "white";
});
Manipulação de elementos DOM

Criar um novo elemento:

Para adicionar um novo elemento ao DOM, use document.createElement e configure o conteúdo com innerText. 


const novoElemento = document.createElement("div");
novoElemento.innerText = "Novo elemento criado!";
document.body.appendChild(novoElemento);

Modificar conteúdo:

Modifique o conteúdo de um elemento existente usando a propriedade innerText.

Area de testes


const elemento = document.getElementById("meu-elemento");
elemento.innerText = "Novo conteúdo!";

Modificar conteúdo usando innerHTML:

Use innerHTML para definir conteúdo HTML dentro de um elemento, incluindo tags HTML.

Cuidado com os riscos de segurança associados. pois pode expor seu aplicativo a ataques de injeção de código (XSS). Certifique-se de que o conteúdo seja seguro

Area de testes


const meuElementoComHTML = document.getElementById("elemento-com-html");
meuElementoComHTML.innerHTML = "Texto em negrito";

Adicionar Novo Elemento Usando appendChild

Usado para colocar algo no final de uma lista. Imagine uma lista, e você quer adicionar um item no final.
Sintaxe: elementoPai.appendChild(novoElementoFilho); Exemplo: Colocar um texto

// Criar um novo elemento
const novoElemento = document.createElement("article");
novoElemento.innerText = "Novo elemento criado!";

// Adicionar o novo elemento como último filho do article atual
document.getElementById("articleAppend").appendChild(novoElemento);

Adicionar Novo Elemento no Iníci Usando prepend

Usado para colocar algo no início de uma lista. Imagine uma lista, e você quer adicionar um item no começo.
Sintaxe: elementoPai.prepend(novoElementoFilho); 

// Criar um novo elemento
const novoElemento = document.createElement("article");
novoElemento.innerText = "Novo elemento criado no início!";

// Adicionar o novo elemento como primeiro filho do article atual
document.getElementById("articlePrepend").prepend(novoElemento);

Remover Elemento

O método remove é usado para remover um elemento do DOM (Document Object Model) em JavaScript. Ele é uma forma simples e direta de remover um nó (elemento) do seu contexto.
Sintaxe: elementoRemovido.remove(); 

// Remover um elemento
const elementoParaRemover = document.getElementById("elementoParaRemover");
elementoParaRemover.remove();

Este elemento pode ser removido clicando no botão "Remove".

Adicionar elementos com .after() e .before()

O método .before() insere conteúdo antes do elemento selecionado. 

                
// Criando um novo item
const newItem = document.createElement('li');
newItem.innerText = 'Novo Item';

// Selecionando o primeiro item existente na lista
const firstItem = document.querySelector('li');

// Inserindo o novo item antes do primeiro item existente
firstItem.before(newItem);

O método .after() insere conteúdo depois do elemento selecionado. 

                
// Criando um novo item
const newParagraph = document.createElement('p');
newParagraph.innerText = 'Novo parágrafo';

// Selecionando o primeiro item existente
const existingParagraph = document.querySelector('p');

// Inserindo o novo item depois do primeiro item existente
existingParagraph.after(newParagraph);

Diferença entre textContent e innerText:

textContentRetorna o conteúdo de texto "puro" do elemento, sem considerar estilos CSS.

innerText Retorna o conteúdo de texto visível do elemento, respeitando estilos CSS, como cores e ocultação.

outerHTML:

A propriedade outerHTML retorna o próprio elemento e seu conteúdo em forma de string, incluindo a tag do elemento. 


const meuElemento = document.getElementById("meu-elemento");
console.log(meuElemento.outerHTML)
Manipulação de atributos DOM

Acessando Atributos

Para obter o valor de um atributo, use
element.getAttribute('nomeDoAtributo') 

                
const link = document.querySelector('a');
const hrefValue = link.getAttribute('href');

Definindo Atributos

para definir ou atualizar o valor de um atributo em um elemento. use
element.setAttribute('nomeDoAtributo', 'valor') 

                
const image = document.querySelector('img');
image.setAttribute('alt', 'Descrição da imagem');

Removendo Atributos

Para remover um atributo de um elemento, Use
element.removeAttribute('nomeDoAtributo') 


const input = document.querySelector('input');
input.removeAttribute('disabled');

Verificação de Atributos

Para verificar se um elemento possui um atributo especifico, Use
element.hasAttribute('nomeDoAtributo') 


const button = document.querySelector('button');
const hasAttribute = button.hasAttribute('disabled');

Manipular classes de CSS

Para manipular classes de um elemento, Para adicionar use
element.classList.add('nomeDaClasse')
ou, Para remover use
element.classList.remove('nomeDaClasse') 


const div = document.querySelector('div');
div.classList.add('destaque');
div.classList.remove('antigo');

Para alternar um classe use
classList.toggle("className")
utilizado para alternar a presença de uma classe em um elemento. Se a classe estiver presente, ela será removida; caso contrário, será adicionada. 


div.classList.toggle('classe-azul');
// Se 'classe-azul' estiver presente, remove; caso contrário, adiciona

Para subistituir uma classe use
classList.replace(oldClass, newClass)
Este método substitui uma classe existente por outra. 


div.classList.replace('antigo', 'novo'); // Substitui 'antigo' por 'novo'
